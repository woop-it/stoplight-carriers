# Changelog 1.6 -> 1.7

### Woop to Carrier

#### Adding barcode to "package.references"

In `POST /quotes`, `POST /deliveries` and `PATCH /deliveries/{deliveryId}`, the node `references` evolves to include a `barcode` for each package involved

<!--
type: tab
title: 1.6.0
-->

```json
{
  ...,
  "packages": [
    ...,
    "quantities": 2,
    "references":[
      "12344567",
      "4YGRVERGEGET"
    ],
    ...
  ],
  ...
}
```
