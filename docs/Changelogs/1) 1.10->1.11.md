# Changelog 1.10->1.11

### Woop to Carrier

#### Adding coordinates to "location" block in the POST/deliveries 

coordinates is composed of two number types :  

- `latitude`
- `longitude`

<!--
type: tab
title: 1.11.0
-->
```json
{
  ...
  "picking": {
    ...
    "location": {
      ...
      "coordinates": {
        "latitude": 50.6405,
        "longitude": 3.0187
      }
    }
  },
  "delivery": {
    "location": {
      ...
      "coordinates": {
        "latitude": 50.62985,
        "longitude": 3.05371
      }
    }
  }
  ...
}

```
<!-- type: tab-end -->

#### Adding volume to "packages" block in the POST/deliveries 

volume of package is defined by `value` and an `unit` in cm3.

<!--
type: tab
title: 1.11.0
-->
```json
{
  ...
  "packages": [
    {
      ...
      "volume": {
        "unit": "cm3",
        "value": 12
      }
    }
    ...
  ]
  ...
}

```
<!-- type: tab-end -->
